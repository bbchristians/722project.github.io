Discipline: Design of Architecture and System Core
The goal of this stage on the Evolutionary Delivery is to build the architecture and system core containing the implementation of the identified risky requirements. The rest of the requirements are implemented in later stages using the output of this stage as base. The test plan of the project is also conceived on this stage.
