Task: Implement the version
The purpose of this task is to add functionality to the existing version until it satisfied all the new requirements. For this the responsible roles should perform the following: Design solution, Code Implementation / Modify Code, Unit / Integration testing, Unit Regression Testing, Code Reviews, Create Build
